# Syntropy Guide — prototype folder

Read in this order:

1. `syntropy-guide-strategy-and-roadmap.md` — the full strategy, taxonomy, and phasing
2. `deployment-guide.md` — exactly how to push this live, step by step
3. `analytics-and-gtm-setup.md` — GA4/GTM setup, how downloads and hub landings are tracked
4. `nav-integration.md` — the homepage nav snippet, and what I need from you to finish it exactly

`syntropy-guide-cms-template.xlsx` is the Google Sheets starter — upload it to Sheets and it becomes the editorial staging area for new entries, plus the source for the homepage's live stats band.

Open `index.html` in a browser to see the working prototype. Every page is self-contained (CSS inlined) so it renders correctly regardless of how the file gets moved or previewed. Move this whole folder as-is into the website repo's root — see the deployment guide for the exact path.
